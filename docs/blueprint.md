# **App Name**: WedEase

## Core Features:

- Wedding Website Builder: Create customizable wedding websites with RSVP management and privacy options.
- Budget Tracker: Track wedding expenses and manage budget allocations.
- Guest List Manager: Manage guest lists, track RSVPs, and organize guest information.
- AI Style Quiz: AI tool that suggests wedding styles based on user preferences.
- Digital Invitations: Design and send digital invitations with RSVP tracking.
- Wedding Registry: Combine gifts from multiple stores into one wedding registry.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to evoke romance and elegance.
- Background color: Light gray (#F0F0F0) for a clean, modern feel.
- Accent color: (red) to highlight key elements and add a touch of luxury.
- Headline font: 'Playfair', serif, for elegant headings. Body font: 'PT Sans', sans-serif, for readability.
- Elegant line icons related to wedding themes (rings, cakes, flowers).
- Clean and modern layout with a focus on user-friendly navigation.
- Subtle animations and transitions to create a smooth user experience.
